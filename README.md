# MosqMK
A R package for analyzing unprocessed data collected by mosquito density monitoring instrument MK 300.

MosqMK是针对由南方医科大学开发的智能化蚊媒监测仪器MK300及相关产品等开发的集“数据整理”模块、“数据可视化”模块和“数据报告”模块工作流为一体的R包。

MK_Data_Collect


MK_Data_Filter

##(2)数据可视化模块
